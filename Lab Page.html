    Lab Assignments

    Lab 1

    This is Lab 1, the first Lab assignment of the semester, unsurprisingly. It was a fun assignment, because it was neat to start getting into working with the HTML. No CSS was used for this assignment

    Lab 2

    This is Lab 2, where I first started to use some style, no CSS yet though. It remained a basic page, but I started trying to separate or break the page up with the help of colors.

    Lab 3

    This is Lab 3, during this assignment I still did not use any CSS, stayed with just using style tags. Though, I added in some images, experimented with text-align, and did some internal linking.

    Lab 4

    This is Lab 4, this the lab I spent a little more time on and incorporated some CSS. Experimented with colors, text styles, making a nav bar, making lists, as well as linking to another (local) page.

    Lab 5

    This is Lab 5, this lab I looked at web design principles. I tried to keep the web page very simple and keep the content condensed. What I really tried to do in this lab was experiment with making a drop-down navagtion bar. I had a lot of trouble getting it to work, but after looking through the web and looking at other mark-up I finally, more or less, got it to work.

    Lab 6

    This is Lab 6, during this lab I created a single html file, then I copied it, and had two individual CSS files for each. I experimented a lot with margins and float for one page of this assignemnt, I worked on putting the images to the left of the text on the page. On the other pages, I spent time making a fixed horizontal navigation bar.

    Lab 7

    For Lab 7 I enhanced the html and css files I created for Lab 6 to do more extensive formatting via css. I experimented using text layouts, selectors, and page formatting.

    Lab 8

    Lab 8 - I created one page about good design principles for mobile web development. I tried to make my page an example using media queries, responsive techniques, and progressive enhancement. To make the page look different on a mobile device compared to a desktop browser. I also created a second simple page to reflect on the Web Accessibility Guidelines.

    Lab 9

    For this lab, Lab 9, I worked with creating a simple form to submit data to a server side script.

    Lab 10

    Lab 10 was used to work with creating an html table to present data. Including formatting and styling the page and the table itself.
