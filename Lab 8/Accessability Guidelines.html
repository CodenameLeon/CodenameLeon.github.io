Accessability Guidlines

Web accessibility means that people with disabilities can use the Web. More specifically, Web accessibility means that people with disabilities can perceive, understand, navigate, and interact with the Web, and that they can contribute to the Web. Web accessibility also benefits others, including older people with changing abilities due to aging.

Web accessibility encompasses all disabilities that affect access to the Web, including visual, auditory, physical, speech, cognitive, and neurological disabilities. The document "How People with Disabilities Use the Web" describes how different disabilities affect Web use and includes scenarios of people with disabilities using the Web.

Techniques to Use

When designing a page you should try to stick with the "industry standards". This helps to knowningly stick with page accessibility, another thing to remember is to use color schemes that are easy to read. Sticking with monochromatic color schemes is an easy way to go about doing this, black text on a white background is easily legiable.
