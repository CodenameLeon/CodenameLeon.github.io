Mobile Design

Mobile Compatibility is huge now in developing a webpage. While most browsing is still done on a computer, browsing on a mobile device still occurs and is increasingly done so. So one must use compatibility techniques such as media queries and responsive techniques to allow the webpage to look good, let alone be usable on a mobile browser.

When you design your page and it is viewed on a mobile device, the page should change to fit the device size, like an image should not get all out of order because it is now being view on a smaller device.
